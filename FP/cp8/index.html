• Es un bloque de código el cual se ejecuta cuando se hace el llamado al mismo.
  • Son bloques de código que realizan una tarea concreta, y estas pueden o no requerir del uso de parámetros.
  • Son procedimientos encapsulados que ejecutan una acción y retornan un resultado.
  • Las funciones permiten la reutilización de código.

IMPORTANTE

Una función tipo void, no retorna ningún valor, únicamente las funciones con tipos de datos primitivos int, char, float, string...

Estructura de una función

Ejemplo

Al crear un función posterior a la función main, se debe de declarar la función antes del main, de lo contrario generará un error.

IMPORTANTE

  • Declaración: Indica el tipo de dato a retornar y el nombre de la misma.
  • Definición: Las instrucciones de código que se ejecutarán.
  • Invocación: El nombre de la función debe ser igual, y si tiene parámetros se debe respetar el tipo de dato y orden de los mismo, utilizar las comas para separar parámetros.

Son funciones que no requieren el uso de parámetros para la ejecución de las instrucciones.

Es un bloque de código que permite ejecutar una tarea en concreto con la característica que para realizar el proceso hace uso de parámetros es decir valores específicos que se solicitan antes de comenzar la ejecución del cuerpo de una función.

La cantidad de parámetros a recibir es indefinida, pero eso dependerá de la tarea que ejecutará y el análisis que se realizó para poder establecer la cantidad de parámetros, los cuales deben ir separados por comas e indicando el tipo de datos que se requiere para cada uno de ellos.
