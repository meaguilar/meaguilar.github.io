Interfaz Gráfica de Usuario, es un intermediario entre código y el usuario, el cual le facilita mediante los diferentes controles hacer uso de la aplicación.

Una graphical user interface o GUI es una interfaz que se puede utilizar para controlar PC, tabletas y otros dispositivos. Las GUI utilizan elementos gráficos como iconos, menús e imágenes para facilitar el manejo del usuario humano. Tanto los sistemas operativos como las aplicaciones utilizan una interfaz gráfica de usuario. De hecho, casi todos los programas para usuarios finales de hoy en día vienen con esta interfaz. Agregar los controles necesarios para crear tu GUI

Aplicación de prefijo a los diferentes controles:

  • frm - Form
  • btn - Button
  • lbl - Label
  • txt - TextBox
  • nud - Numeric up down
  • chk - CheckBox
  • rad - RadioButton
  • grp - GroupBox
  • pic - PictureBox
  1. Crea una aplicación para escritorio en Rider en esta carpeta
  2. Selecciona una GUI que deseas recrear, buscala en la web
  3. Toma la captura de pantalla de esa GUI y colócala dentro del proyecto
  4. Tu GUI debe contar con los siguientes controles: label, botones, picture, textbox como mínimo y todos los que consideres necesarios
  5. Tu GUI debe contar con el evento clic
  6. Ingresa a evaluaciones en Moodle y clic en cp3
  7. Busca tu nombre y acepta la tarea en GitHub Classroom
  8. Clona el repositorio [git clone <url>]
  9. Abre el proyecto y cambia la versión de .NET framework, por la que tengas instalada en tu computadora.
  10. Desarrolla la solución
  11. Leer buenas prácticas https://meaguilar.github.io/POO/Evaluaciones/cp1/#6
  12. Utiliza buenas prácticas para realizar el commit [git commit -m ""]
