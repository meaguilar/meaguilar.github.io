Visualización de la información a través de la GUI
  1. Ingresa a evaluaciones en Moodle y clic en cp5
  2. Clona el repositorio [git clone <url>]
  3. Utiliza Azure Data Studio o SQL Management Studio, para conectarte a la BD.
  4. Conectate a la base de datos que se utilizó en el parcial de Bases de Datos, es decir al servidor 192.168.233.123,5225, BD_Parcial_II e ingresa tu usuario y contraseña.
  5. Crea una consulta sin parámetros y una consulta con parámetros.
  6. Crea un proyecto para escritorio
  7. Configurar la cadena de conexión en los recursos de tu proyecto.
  8. Crea una GUI con los controles que consideres necesarios que permita mostrar la información de las consultas que realices a la BD.
  9. Leer buenas prácticas https://meaguilar.github.io/POO/Evaluaciones/cp1/#6
  10. Utiliza buenas prácticas para realizar el commit [git commit -m ""]
  11. Envía tus cambios al repositorio [git push]
