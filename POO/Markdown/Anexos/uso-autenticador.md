
# Configuración de autenticador

  

## Autenticador

Microsoft Autofill es un administrador de contraseñas desarrollado por Microsoft.​​ Es compatible con múltiples plataformas, como Android, iOS y el navegador Google Chrome. Es parte de la aplicación Microsoft Authenticator en Android e iOS, ​ y una extensión del navegador en Google Chrome. ​

  

### Usando autenticador de Microsoft

  

1. Descargar en google playStore [Microsoft Authenticator App](https://play.google.com/store/apps/details?id=com.azure.authenticator)

![enter image description here](https://raw.githubusercontent.com/meaguilar/meaguilar.github.io/refs/heads/main/POO/imagenes/uso-autenticador/1.jpeg)

2. Abrir la applicación y agregar el correo institucional.

![enter image description here](https://raw.githubusercontent.com/meaguilar/meaguilar.github.io/refs/heads/main/POO/imagenes/uso-autenticador/2.jpeg)

![enter image description here](https://raw.githubusercontent.com/meaguilar/meaguilar.github.io/refs/heads/main/POO/imagenes/uso-autenticador/3.jpeg)

3. Una vez agregado el correo institucional puedes usar el codigo Qr para ir agregando mas cuentas. 
Ejemplo: Agregar la cuenta de Github, dar click al icono de código Qr y se abrirá un escáner.

![enter image description here](https://raw.githubusercontent.com/meaguilar/meaguilar.github.io/refs/heads/main/POO/imagenes/uso-autenticador/4.jpeg)

![enter image description here](https://raw.githubusercontent.com/meaguilar/meaguilar.github.io/refs/heads/main/POO/imagenes/uso-autenticador/5.jpeg)


4. Ir a Github , buscar el código Qr para poder escanear en el autenticador, una vez agregado a la aplicación, escribir el código para verificar que se ha configurado bien la cuenta de Github en el autenticador.
   

> 💡****Nota:**** En Github, tienes que ir a **tu perfil>settings>password and authentication**
